# `jules-tasks` Skill

Use this skill to audit Jules sessions.

1. `node .agent/scripts/jules-batch-review.js` (list tasks)
2. `node .agent/scripts/jules-extract-patch.js <id>` (extract patch)
3. `node .agent/scripts/jules-api.sh approve-plan <id>` (approve)
